# GeoEarn — Инвестиционный питч

> Telegram Mini App: геймифицированная платформа лояльности для офлайн-бизнесов  
> Рынок: Узбекистан → СНГ → Telegram-аудитория 950 млн пользователей

---

## Что такое GeoEarn

**GeoEarn** — это мобильная экосистема внутри Telegram, где пользователи получают **GEO-токены** за посещение партнёрских заведений (кафе, рестораны, магазины, салоны). Бизнесы платят за реальный трафик — не за клики или показы.

**Ключевая идея:** офлайн-чекин через QR-код → мгновенное начисление GEO → обмен на реальные деньги.

---

## Проблема, которую мы решаем

| Боль бизнеса | Боль пользователя |
|-------------|-----------------|
| Дорогая digital-реклама без гарантии посещений | Скидочные карты, которыми никто не пользуется |
| Нет инструмента для создания повторных визитов | Нет мотивации возвращаться в то же место |
| Сложные системы лояльности требуют интеграций | Разные приложения для каждого бренда |
| Невозможно измерить ROI офлайн-рекламы | Кэшбэк приходит через 2 недели |

**Решение GeoEarn:** единая платформа в Telegram — без установки приложений, QR-код за 3 секунды, мгновенные выплаты.

## Бизнес-модель

### Доходы платформы

**1. Комиссия с кампаний (10% upfront)**
```
Бизнес создаёт кампанию на 1 000 GEO →
Платформа сразу получает 100 GEO →
Остальные 900 GEO расходуются по мере чекинов
```
Комиссия фиксируется в базе атомарно и сразу зачисляется в `platform_wallet`.

**2. Спред при выводе**
- Пользователи выводят GEO по курсу платформы (GEO→UZS)
- Курс устанавливается администратором
- Платформа контролирует спред

**3. Промо-охота (Promo QR Hunt)**
- Платные промо-кампании с токенами редкости (common / rare / epic / legendary)
- Геолокационное ограничение → трафик к конкретной точке
- Отдельный монетизационный канал без участия регулярного бизнеса

### Юнит-экономика (пример)

| Показатель | Значение |
|-----------|----------|
| Средний бюджет кампании | 5 000 GEO |
| Комиссия платформы (10%) | 500 GEO |
| Средний курс | 1 GEO = 1 000 UZS |
| Доход с кампании | 500 000 UZS (~40 USD) |
| Кампаний в месяц (10 бизнесов) | 30 |
| Выручка в месяц | 15 000 000 UZS (~1 200 USD) |

## Что уже реализовано

### Продукт готов к запуску ✅

#### Для пользователей
- [x] **Чекин через QR** — сканирование, геоверификация, мгновенное начисление
- [x] **GEO-кошелёк** — баланс, история транзакций
- [x] **Вывод средств** — заявка на вывод GEO → UZS на карту
- [x] **Онбординг** — приветственный флоу с объяснением системы
- [x] **Интерактивная карта** — все партнёры на Leaflet-карте

#### Геймификация (полностью рабочая)
- [x] **10 уровней** (Novice → Legend) с множителями наград 1.0×–1.25×
- [x] **Дейли стрики** — streak с бонусами на 7/14/30 день (500 / 1 500 / 5 000 GEO)
- [x] **12 заданий** — ежедневные, еженедельные, единоразовые
- [x] **6 достижений** — за реальное поведение (стрик 30 дней, 10 заведений, etc.)
- [x] **Буст-события** — временные множители (до 1.5×) для акций
- [x] **Стрик-заморозка** — 1 пропуск в неделю без потери серии

#### Реферальная система
- [x] **Уникальные ссылки** для каждого пользователя
- [x] **Приветственный бонус**: 1 000 GEO рефереру + 500 GEO новому пользователю
- [x] **Пассивный доход**: 5% от каждого чекина приглашённого в течение 30 дней
- [x] Антидублирование: идемпотентность на уровне БД по visit_id

#### Для бизнеса
- [x] **Личный кабинет** в Mini App — без сторонних инструментов
- [x] **Создание кампаний**: бюджет → автоматический расчёт награды и лимита
- [x] **PIN-верификация** — защита от фиктивных чекинов (6 цифр, 15 мин)
- [x] **QR-коды кампаний** — уникальный токен на каждую акцию
- [x] **Статистика** — сегодня / 7 дней / предыдущие 7 дней
- [x] **Пополнение баланса** через оператора
- [x] **Авто-остановка** — кампания останавливается при нехватке баланса

#### Платформа (SuperAdmin)
- [x] **Dashboard** — общая выручка, активные пользователи, объём выводов
- [x] **Управление 5 000 +** пользователей: бан, разбан, коррекция баланса
- [x] **Ручное одобрение выводов** с аудит-логом каждого действия
- [x] **Fraud-детекция** — подозрительная активность, мультичекины
- [x] **Управление курсом** GEO→UZS с историей изменений
- [x] **Промо QR Hunt** — создание/редактирование/аналитика промо-кампаний
- [x] **Платформенные кампании** — кампании за счёт накопленных комиссий
- [x] **Управление задачами и достижениями** из интерфейса

#### Технический фундамент
- [x] HMAC-SHA256 аутентификация (стандарт Telegram Mini App)
- [x] 10 атомарных хранимых процедур PostgreSQL (нет race conditions)
- [x] 20 таблиц с полным аудит-логом
- [x] Telegram-бот с 6 хэндлерами и 5 фоновыми задачами
- [x] Rate limiting на всех уязвимых эндпоинтах
- [x] Railway + Supabase (zero-downtime деплой)
